20 Git Commands and Their Functions:
1. `git init`: Initializes a new Git repository.
2. `git clone [repository]`: Clones a repository into a new directory.
3. `git add [file]`: Adds changes in the working directory to the staging area.
4. `git commit -m "[message]"`: Records changes to the repository with a descriptive message.
5. `git status`: Shows the status of changes as untracked, modified, or staged.
6. `git log`: Displays a log of commits.
7. `git branch [branch name]`: Creates a new branch.
8. `git checkout [branch name]`: Switches to the specified branch.
9. `git merge [branch name]`: Merges changes from the specified branch into the current branch.
10. `git pull`: Fetches changes from a remote repository and merges them into the current branch.
11. `git push`: Pushes changes to a remote repository.
12. `git remote -v`: Lists all remote repositories.
13. `git diff`: Shows changes between commits, commit and working tree, etc.
14. `git reset [file]`: Unstages changes for the specified file.
15. `git stash`: Temporarily saves changes that are not ready to be committed.
16. `git fetch`: Fetches changes from a remote repository without merging.
17. `git tag [tag name]`: Adds a tag to the current commit.
18. `git remote add [name] [URL]`: Adds a remote repository.
19. `git rm [file]`: Removes a file from the working directory and stages the deletion.
20. `git clean -fd`: Removes untracked files from the working directory.
